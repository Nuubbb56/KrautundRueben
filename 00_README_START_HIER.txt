Kraut & Rueben - Komplettes Abschluss-/Abgabe-/Praesentationspaket (FINAL, C# 8 kompatibel)

Dieses Paket ist auf C# 8.0 und .NET 8 Windows Forms angepasst.

Wichtige Punkte:
- Im Hauptordner liegen jetzt KrautUndRueben.sln und KrautUndRueben.slnx.
- Die Projektdatei nutzt <LangVersion>8.0</LangVersion>.
- Die Datenbankstruktur ist naeher an der 3. Normalform: Adressen und Statuswerte sind ausgelagert, Gesamtbetraege werden aus Bestellpositionen berechnet.
- Die App nutzt Views und Stored Procedures, damit die Demo-Spalten trotzdem uebersichtlich bleiben.

INHALT
1. SQL/01_KrautUndRuebenDB.sql
2. KrautUndRueben.sln / KrautUndRueben.slnx
3. VisualStudioProjekt/KrautUndRuebenApp/ (C#-Projekt)
4. Doku/
5. Praesentation/

Start:
1) SQL/01_KrautUndRuebenDB.sql in SQL Server ausfuehren.
2) KrautUndRueben.sln oder KrautUndRueben.slnx in Visual Studio oeffnen.
3) Falls noetig Verbindung in Data/Database.cs oder per Umgebungsvariable KRAUT_UND_RUEBEN_CONNECTION anpassen.
4) Projekt starten.
